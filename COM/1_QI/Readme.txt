Приложение предназначено для работы с компонентом - обработка выборки (на вход принимаются 2 массива и по формулам вычисляются выборочное среднее, выборочная дисперсия и исправленная выборочная дисперсия)

Взаимодействие с компонентом server осуществляется при помощи интерфейсов IGet_Array, который предназначен для чтения массивов из текстовых файлов, и ISample_Processing, в котором присутствуют методы вычисления определенных оценок изменчивости.
